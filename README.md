# Feedback-Survey
Here a feedback survey is implemented using HttpSession.By using HttpSession to implement a feedback survey can be a useful tool for gathering feedback and improving your products or services based on user feedback. By using HttpSession to store survey data, you can easily collect and analyze user feedback without needing to store data in a database or other external storage medium.
